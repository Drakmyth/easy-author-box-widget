== Description ==

Easy Author Box Widget adds a widget to display a simple configurable author box with social icons.

== Installation ==

To install this plugin:

1. Upload the `easy-author-box-widget` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's it!

Alternatively you can search for the plugin from your WordPress dashboard and install from there.
